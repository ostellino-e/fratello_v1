# Fratello v0.3

Mejora del Punto 1:

- Producción estibada/realizada.
- Día desplegable lunes a domingo.
- Producción base automática por día.
- Botón para cambiar producción base.
- Producción especial/por encargo en sección plegable.
- Mejor uso en celular.

## Uso
1. Abrir `index.html`.
2. Elegir día de producción.
3. Cambiar producción base una sola vez si hace falta.
4. Guardar producción realizada.
5. Pegar pedidos y calcular diferencias.
